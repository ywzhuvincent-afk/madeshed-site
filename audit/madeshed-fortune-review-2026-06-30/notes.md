## 流程步骤

1. 首页
   - 健康度：中上
   - 优点：定位清楚，“命盘 × 数据”有差异化，风险提示比普通算命产品克制。
   - 风险：仍有 Beta/内测气质；首页 ¥299 一次性买断，与 Daily 页 PRO ¥49/月 不一致。

2. 生成命盘
   - 健康度：中
   - 优点：四柱展示清晰，适合非专业用户快速理解。
   - 风险：出生地文案写“用于精确计算真太阳时”，但当前主站算法没有把城市经度/时区用于 `calcBazi`。农历/公历切换也没有看到参与计算。

3. 仪表盘
   - 健康度：中
   - 优点：把日元、强弱、大运、今日评分放在一屏，产品价值容易被理解。
   - 风险：同一测试命盘在 Dashboard 显示“中和 58”，完整排盘页显示“偏强 58”，专业用户会认为口径不统一。

4. 每日评分
   - 健康度：中
   - 优点：记录入口很顺，适合做个人交易复盘。
   - 风险：“你的历史”当前来自模拟数据，容易被误认为真实个人回测。命理评分主要按流日天干/五行固定打分，未充分纳入地支、刑冲合害、空亡、原局结构、大运流年与流月的综合关系。

5. 完整排盘
   - 健康度：中上
   - 优点：四柱、十神、藏干、纳音、空亡、神煞、大运流年流月流日层级比较完整。这个页面比 Dashboard 的命理算法更接近专业排盘。
   - 风险：页面声称“四柱、十神、藏干、纳音、空亡、大运、流年、流月均由排盘引擎精确计算”，但用神仍是近似自动判定，建议把“精确”限定在排盘层，不用于断法层。

## 命理准确性结论

- 基础排盘：对公历、北京时间、普通时辰场景基本可靠，依赖 `lunar-javascript` 比手写干支表更稳。
- 出生地/真太阳时：当前是最大准确性缺口。海外用户、边界时辰、子时附近、夏令时场景都可能出错。
- 旺衰用神：目前属于入门级自动判定，不是专业师傅级断法。完整排盘页比 Dashboard 更好，但两套口径需统一。
- 大运流年：排盘展示可以用；吉凶评分仍偏模型化，不能称专业断运。
- 投资财运：可以作为“行为复盘标签”，不适合作为“可预测市场胜率”的命理结论。

## 建议优先级

1. 先修准确性底座：真太阳时、时区、农历输入、早晚子时、城市选择真正参与排盘。
2. 统一 Dashboard 和完整排盘页的强弱/用神算法。
3. 把“个人历史/回测”在无真实数据时明确标成“示例数据”。
4. 日评分从固定 78/60/42 升级为可解释的分层评分：原局强弱、大运、流年、流月、流日、支冲合害、十神、空亡分别给权重。
5. 明确“排盘准确”和“断法近似”两类说明，避免专业用户抓住一句“精确”质疑全站。
